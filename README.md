# Portfólio PHP - Rocketseat

Este projeto foi desenvolvido como parte do curso de formação PHP da Rocketseat. Trata-se de um portfólio pessoal, proposto como atividade prática para aplicar os conhecimentos adquiridos ao longo do curso.

## Sobre o Projeto

O objetivo deste projeto é apresentar um portfólio moderno, responsivo e funcional, utilizando PHP para estruturar as páginas e componentes, além de recursos de estilização com CSS e Tailwind.

## Estrutura
- **index.php**: Página principal do portfólio
- **components/**: Componentes reutilizáveis em PHP
- **assets/**: Imagens e outros recursos estáticos
- **utils/**: Utilitários e variáveis globais
- **style.css**: Estilos personalizados

## Objetivo

Demonstrar habilidades em PHP, organização de projetos e boas práticas de desenvolvimento web, conforme proposto pela Rocketseat.
